## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/barry-williams/tower-of-colour** and import

## Edit this project ![Build status badge](https://github.com/barry-williams/tower-of-colour/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/barry-williams/tower-of-colour** and click import
